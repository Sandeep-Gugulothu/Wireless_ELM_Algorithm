# Application of Deep Learning in Wireless Communication
Research and implementation of intelligent signal detection and channel estimation for MIMO-OFDM systems.

## Project Overview
This project explores the integration of Autoencoders (AE) and Extreme Learning Machines (ELM) to improve detection accuracy and reduce computational complexity in wireless networks.

## Repository Structure
- `Latex Format For Term Paper.tex`: Core research manuscript.
- `ae_elm_detection.py`: Initial implementation of the detection algorithm.
- `Images/`: Simulation results and architecture diagrams.
